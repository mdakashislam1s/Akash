# Perfect Todo Web App

A polished, fast, and fully client-side todo application built with vanilla HTML, CSS, and JavaScript.

## Features

- Add, complete, edit, and delete tasks
- Filter tasks by **All / Active / Completed**
- Clear all completed tasks in one click
- Persistent storage with `localStorage`
- Live stats: remaining tasks and completion percentage
- Responsive layout for desktop and mobile
- Light/Dark theme toggle with saved preference
- Keyboard-friendly and accessible controls

## Run locally

No build step is required.

1. Open `index.html` directly in your browser, **or**
2. Serve the directory with a static server:

```bash
python3 -m http.server 4173
```

Then visit: <http://localhost:4173>
